* Projet pluridisciplinaire
Protocole de communication série
Xavier Galzin, Romain Désille, Stanislas Bertrand, Frédéric Meslin

Caractéristiques principales :

Configuration : 8 bits, pas de parité, 1 bit de stop
		Baudrate variable

Notation : x à renseigner
	   U état non précisé

Trames génériques :

* Trame de contrôle :
Trame : typ r/w log ad4 ad3 ad2 ad1 ad0 <8 bits>
	 0   0   0   x   x   x   x   x

-> typ : type de trame (1 : contrôle | 0 : données)
-> r/w : type de transfert (0 : lecture | 1 : écriture)
-> log : type de données envoyées (0 : entiers 16 bits | 1 : chaines de caractères)
-> (ad4 ... ad0) : adresse de la valeur concernée (0 -32)

* Trame de données entières :
  MSB : typ b06 b05 b04 b03 b02 b01 b00 <8 bits>
	 0   0   0   0  d15 d14 d13 d12

  MSB : typ b06 b05 b04 b03 b02 b01 b00 <8 bits>
	 0   0   0   0  d11 d10 d09 d08

  LSB : typ b06 b05 b04 b03 b02 b01 b00 <8 bits>
	 0   0   0   0  d07 d06 d05 d04

  LSB : typ b06 b05 b04 b03 b02 b01 b00 <8 bits>
	 0   0   0   0  d03 d02 d01 d00


-> typ : type de trame (0 : contrôle | 1 : données)
-> (d15 ... d00) : valeur de la donnée 16 bits

* Trame de données chaine de caractères :
Char1 : typ da6 da5 da4 da3 da2 da1 da0 <8 bits>
	 0  ch6 ch5 ch4 ch3 ch2 ch1 ch0

Char2 : typ da6 da5 da4 da3 da2 da1 da0 <8 bits>
	 0  ch6 ch5 ch4 ch3 ch2 ch1 ch0

	...

Charn : typ da6 da5 da4 da3 da2 da1 da0 <8 bits>
	 0  ch6 ch5 ch4 ch3 ch2 ch1 ch0

	...

  FIN : typ b06 b05 b04 b03 b02 b01 b00 <8 bits>
	 0   0   0   0   0   0   0   0

-> typ : type de trame (1 : contrôle | 0 : données)
-> (ch6 ... ch0) : caractère table ASCII individuel
Le paquet end termine la transmission de la chaine

Exemples de trames :

* Lecture de données
Trame : typ r/w log ad4 ad3 ad2 ad1 ad0 <8 bits>
	 1   0   0   x   x   x   x   x
-> Demande de lecture d'une valeur

Réponse immédiate : une trame de données entières

* Ecriture de données
Trame : typ r/w log ad4 ad3 ad2 ad1 ad0 <8 bits>
	 1   1   0   x   x   x   x   x
Suivie d'une trame de données entières
-> Demande d'écriture de données + donnée

* Envoi d'un message de log
Trame : typ r/w log ad4 ad3 ad2 ad1 ad0 <8 bits>
	 1   U   1   U   U   U   U   U
Suivie d'une trame de chaine de caractères
